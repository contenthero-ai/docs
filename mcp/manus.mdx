---
title: "Manus"
description: "Connect ContentHero to Manus as a custom MCP connector."
---

Manus supports custom MCP servers, so your Manus agents can use ContentHero directly.

## Add the connector

<Steps>
  <Step title="Open MCP connectors">
    In Manus, open the integrations settings and choose to add a custom MCP connector. See the [Manus MCP connectors docs](https://manus.im/docs/integrations/mcp-connectors) for the current location.
  </Step>
  <Step title="Enter the server URL">
    Set the server URL:

    ```
    https://mcp.contenthero.ai
    ```
  </Step>
  <Step title="Authorize">
    Complete the OAuth flow. Sign in with your ContentHero account and approve. No API key needed.
  </Step>
</Steps>

## Verify

Ask your Manus agent to read your ContentHero balance. It calls `get_balance` and returns your credits. Once connected, Manus can run the full pipeline, including scheduled posts, through ContentHero.
